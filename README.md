# 🎵 YouTube to MP3 Converter

A simple and powerful tool to convert YouTube videos into MP3 files — with support for search, thumbnails, and batch downloads.

---

# 🚀 Quick Start (Super Simple)

Follow these steps and you’ll be up and running in minutes 👇

---

## 🧰 1. Install Node.js

You need Node.js to run the app.

👉 Go here: https://nodejs.org  
👉 Download **LTS version** (recommended)  
👉 Install it (just click Next → Next → Finish)

✅ To check it worked, open Terminal / Command Prompt and type:

```bash
node -v
npm -v

```

then where you find package.json type

```bash
npm i

```

locate app.js file and type

```bash
node app.js
```
